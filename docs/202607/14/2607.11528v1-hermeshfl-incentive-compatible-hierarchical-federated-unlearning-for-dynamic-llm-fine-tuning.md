## 摘要
针对大语言模型（LLM）微调的分层联邦遗忘学习（HFUL）面临分层聚合、动态客户端参与以及LLM适配中强参数耦合等重大挑战。由于模型更新在多个聚合阶段传播，而遗忘请求可能与客户端离开和重新加入同时发生，选择性地移除客户端贡献尤为困难。为解决这些问题，我们提出**HermesHFL**，一个支持选择性遗忘、动态客户端参与和客户端重新集成的分层联邦学习框架，通过基于LoRA的参数高效微调（PEFT）实现可扩展的LLM微调。我们构建了一个统一的优化问题，在异构客户端行为下联合建模客户端参与、边缘关联、激励分配和遗忘。为高效求解该问题，我们开发了**Neogen**，一个神经引导的双层进化优化框架，结合CMA-ES进行连续激励优化，并利用基于CHC的进化机制进行离散的参与和关联决策。神经代理模型进一步加速优化并提高搜索效率。在LLM微调任务上的大量实验表明，HermesHFL在模型效用、遗忘有效性、收敛稳定性和资源效率方面始终优于最先进的基线方法。

## Abstract
Hierarchical federated unlearning (HFUL) for large language model (LLM) fine-tuning faces significant challenges due to hierarchical aggregation, dynamic client participation, and strong parameter coupling in LLM adaptation. Selectively removing client contributions is particularly difficult because model updates propagate across multiple aggregation stages while unlearning requests may coincide with client departures and rejoining. To address these issues, we propose **HermesHFL**, a hierarchical federated learning framework that supports selective unlearning, dynamic client participation, and client reintegration for scalable LLM fine-tuning via parameter-efficient fine-tuning (PEFT) with LoRA. We formulate a unified optimization problem that jointly models client participation, edge association, incentive allocation, and unlearning under heterogeneous client behaviors. To solve this problem efficiently, we develop **Neogen**, a neural-guided bilevel evolutionary optimization framework that combines CMA-ES for continuous incentive optimization with a CHC-based evolutionary mechanism for discrete participation and association decisions. A neural surrogate further accelerates optimization and improves search efficiency. Extensive experiments on LLM fine-tuning tasks demonstrate that HermesHFL consistently outperforms state-of-the-art baselines in model utility, unlearning effectiveness, convergence stability, and resource efficiency.

## 1. 核心问题与研究背景

- **研究背景**：大语言模型（LLM）微调逐渐从集中式走向层次化联邦学习（HFL），以满足隐私保护和通信效率的要求；同时，GDPR 等法规的“删除权”催生了**联邦遗忘学习（FUL）**的需求。
- **核心挑战**：
  - 层次化架构中，模型更新经多层聚合传播，移除特定客户端影响时，容易破坏全局模型效用。
  - 客户端行为动态变化，可能离开、请求遗忘后又重新加入（leave–unlearn–rejoin），使遗忘与聚合一致性更难保障。
  - 缺乏有效的**激励机制**来协调云‑边缘‑客户端三层之间的利益，保证参与稳定性和遗忘有效性。
- **整体含义**：论文旨在构建一个**支持选择性遗忘、动态客户端重新加入且激励兼容**的层次化联邦遗忘框架，以满足动态 LLM 微调下的隐私和实用性需求。

## 2. 方法论

- **HermesHFL 框架设计**
  - **三层角色**：总统（云服务器）下发任务并分配总预算；管理者（边缘服务器）负责选工人、签合同、边缘聚合；工人（客户端）执行本地 LoRA 微调并可请求遗忘/重新加入。
  - **参数高效微调**：采用 LoRA 技术，仅更新低秩矩阵 \(A, B\)，大幅降低通信和计算开销。
  - **遗忘机制**：请求遗忘的工人执行**梯度上升（GA）**，最大化自身数据的损失，反向削弱已学知识；用 KL 散度验证遗忘程度，达到阈值后允许工人刷新属性并重新加入。
  - **合同与激励模型**：合同包含支付 \(p\)、遗忘惩罚 \(q\)、目标质量 \(\theta\)；工人属性涵盖数据分布、算力、通信、隐私成本和信用分（基于历史遗忘频率），刻画个体差异与不确定性。
  - **预算与目标对齐**：总统按管理者关联工人的贡献质量比例分配新预算，从而将三层优化简化为**双层优化**（管理者‑工人联合优化），避免冗余层次。

- **双层优化问题建模**
  - **上层**：工人最大化自身总效用（收入 − 成本 + 遗忘收益 − 惩罚）。
  - **下层**：管理者选择工人和合同，最大化总效用（模型质量收益 − 支付给工人的成本）。
  - 约束包括二进制选择变量、预算限制、个人理性、质量阈值等，形成混合整数非线性非凸问题。

- **Neogen 神经进化优化算法**
  - **上层**：采用 **CMA‑ES**（协方差矩阵自适应进化策略）采样连续变量 \(p, q\)，根据 top‑K 精英更新均值、步长和协方差矩阵，适应连续搜索空间且无需梯度。
  - **下层**：采用 **CHC** 进化算法处理二进制变量（工人选择 \(x\)、管理者关联 \(y\)）。引入半均匀交叉（HUX）、禁止近亲繁殖（Hamming 距离阈值）、灾难性变异等机制，保证离散组合空间的多样性和全局探索。
  - **神经网络代理加速**：训练一个 3 层 MLP，从 (上层决策) → (下层最优解) 的映射。早期通过精确 CHC 求解积累数据；后期利用 NN 预测初始解，再在限定 Hamming 邻域内局部 CHC 细化，大幅减少下层适应度评估次数。

## 3. 实验设计

- **模型与数据**：
  - 基座模型：GPT‑2 (124M)，LoRA 秩 \(r=8, \alpha=16\)，目标模块为 \(c_{attn}, c_{proj}\)。
  - 数据集：SST‑2（二分类情感分析）和 AGNews（4 分类主题分类），均为文本分类。
- **联邦配置**：
  - 规模 1：10 个工人 + 3 个管理者（\(β^{sum}=30\)）
  - 规模 2：20 个工人 + 4 个管理者（\(β^{sum}=60\)）
  - 每工人 200‑1000 条非独立同分布样本，全局轮次 \(G=6\)，边缘轮次 \(E=5\)。
- **对比基准（优化机制消融）**：
  - Pure EA：去掉神经代理的 CMA‑ES+CHC。
  - Simulated Annealing (SA)：200 次迭代几何冷却。
  - Greedy：基于 \(\theta / cost\) 比贪婪选择。
  - Random：随机选择和固定支付策略。
- **遗忘策略消融**：
  - Rejoin：满足 KL 阈值后刷新属性重新加入。
  - No Rejoin：永久移除该工人。
  - Retrain：完全重置模型无该工人重新训练。
- **评价指标**：全局测试准确率、管理者效用、工人效用、每轮适应度评估次数、总耗时。

## 6. 主要结论

- HermesHFL 通过**联合优化选人、关联、激励和遗忘**，在 LLM 微调任务上实现了**高准确率、正向管理者和工人效用平衡**。
