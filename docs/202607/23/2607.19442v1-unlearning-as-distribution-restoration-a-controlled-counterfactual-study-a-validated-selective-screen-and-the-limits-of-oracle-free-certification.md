## 摘要
机器遗忘通常通过在训练探针上匹配重训练参考模型来评估。在一个包含匹配重训练参考的受控临时事实测试平台上，我们发现这一标准可能有利于保留留存知识的方法：其评为合格的方法在遗忘事实上的得分比从未学习水平低-2.82纳特（聚类置信区间[-3.16,-2.48]）。我们将遗忘重新定义为恢复到匹配参考，并审计了45个模型-种子单元格（涵盖五个开放架构家族）上的无参考筛选和类似证书的标准。参考自身证伪了绝对保留/往返证书：注入模型（其构建即保留保留集）在41/45个单元格中未通过固定保留阈值，在31/45个单元格中未通过自身往返测试，并且参考仅在1/45个单元格中完全认证。基于基线的留存筛选作为选择性必要测试依然稳健：在密封挑战集上，它在45/45个单元格中拒绝了注入模型，在44/45个单元格中接受了参考，并部分检测到实体路由抑制（35/45）；这是一项具有测量敏感性的必要测试，而非充分性证书。以参考自身工作点为锚点的损伤相对重新校准，在15/45个单元格中认证了一小部分；在不弃权的情况下，其选择在优化轴上处于重训练噪声范围内（0.80纳特），而常见的训练探针标准则偏离5.17纳特（这是一项支持性比较，而非直接对标）。固定幅度logit抑制攻击在12/45个单元格中击败了整个前向测试电池，因此仅基于前向的认证是不健全的；我们的方法是一种针对实际生产方法的经验性选择性测试。可识别性定理界定了哪些事实能够允许无参考遗忘阈值，TOFU被预测为边界案例。

## Abstract
Machine unlearning is commonly evaluated by matching a retrained oracle on trained probes. In a controlled nonce-fact testbed with a matched retraining reference, we find this criterion can favor methods that retain held-out knowledge: candidates it rates adequate score held-out forget facts $-2.82$ nats below the never-learned level (cluster CI $[-3.16,-2.48]$). We recast unlearning as restoration to the matched reference and audit oracle-free screens and certificate-style criteria across 45 model-seed cells spanning five open architecture families. The reference itself falsifies an absolute retain/round-trip certificate: the injected model, which retains the retain set by construction, fails the fixed retain threshold in 41/45 cells and its own round trip in 31/45, and the reference fully certifies in only 1/45. A base-anchored held-out screen remains strong as a selective necessary test: on a sealed challenge suite it rejects the injected model in 45/45 cells, accepts the reference in 44/45, and partially detects entity-routing suppression (35/45); it is a necessary test with measured sensitivity, not a sufficiency certificate. A damage-relative recalibration anchored to the reference's own operating point certifies a small subset in 15/45 cells; where it does not abstain, its picks lie within retraining noise (0.80 nats) on the axes it optimizes, while the common trained-probe criterion sits 5.17 nats away (a supporting comparison, not a head-to-head benchmark). A fixed-magnitude logit-suppression attack defeats the full forward battery in 12/45 cells, so forward-only certification is not sound; our method is an empirical selective test for methods-as-produced. An identifiability theorem delimits which facts admit an oracle-free forget threshold at all, with TOFU as the predicted boundary case.

## 论文详细总结（自动生成）

### 1. 核心问题与整体含义
- **研究动机**：当前机器遗忘（machine unlearning）的主流评估方法是，在“训练探针”（trained probes）上计算遗忘模型与完整重训练模型（oracle）的匹配度。但作者发现，这一标准存在系统性偏差：它可能将“仅抑制表面输出、实则仍保留内部知识”的方法误判为合格。
- **核心问题**：如何在没有完整重训练模型（oracle-free）的条件下，可靠地区分“真正恢复至未学习分布的遗忘”与“仅做表面答案抑制的伪遗忘”？
- **整体含义**：论文将遗忘重新定义为**分布恢复**（distribution restoration），即遗忘模型应恢复到匹配的重训练参考分布。作者通过构建一个**匹配的反事实重训练参考**，审计现有标准和自身新提出的无参考筛选/认证方法，揭示现有标准的缺陷，界定无参考认证的可行边界，并最终将遗忘认证定位为一种**经验性、可放弃的选择性测试**，而非具有绝对保证的证书。

### 2. 方法论
- **受控反事实测试平台**
  - 从基模型\(M_0\)出发，通过继续预训练同时注入一组临时（nonce）遗忘事实集\(F\)和保留事实集\(R\)，得到注入模型\(M_{inj}\)。
  - 匹配的重训练参考模型\(M_{oracle}\)仅注入\(R\)，从未见过\(F\)，构建出“如果从未学习过\(F\)”的反事实状态。该参考模型是评估的黄金标准，但通常不可用。
- **无参考（oracle-free）往返认证**
  - **核心思路**：对一个遗忘候选模型\(C\)，重新微调它去学习遗忘集\(F\)（即“再获取”），然后测量再获取后模型与\(M_{inj}\)的KL散度（在\(R\)和通用文本上），得到**往返残余**\(\text{cert}(C)\)。真正的分布恢复应当能“闭环”：再获取后应接近\(M_{inj}\)；而单纯抑制的方法则会留下永久性附带损害，无法复原。
  - **遗忘资格筛查**：使用**基线锚定的保留外检测**（base-anchored held-out screen）。对候选模型\(C\)，计算其在\(F\)的保留外改写上的NLL基差\(\Delta_{F_i} = \text{NLL}_C(F_i) - \text{NLL}_{M_0}(F_i)\)，并与一组**从未学习过的探针事实**\(P\)的基差\(\Delta_{P_j}\)进行单边Mann-Whitney U检验。若\(\{\Delta_{F_i}\}\)显著低于\(\{\Delta_{P_j}\}\)，则排除该候选（说明仍保留遗忘知识）。锚定至\(M_0\)避免全局损伤伪装成遗忘。
  - **可识别性限制（定理）**：形式化地证明了，仅当oracle的遗忘水平\(\tau^*\)可从无oracle可观测量（如从未学习探针的分布）恢复时，无参考选择才可能。这解释了为什么TOFU这类可推断事实无法进行无参考认证。
- **损害相对重新校准（damage-relative recalibration）**
  - 放弃固定阈值，转而以**匹配参考模型自身的工作点**和**独立重训练重抽样引入的复制噪声**作为容差。候选模型需满足四项条件：通过基线锚定筛查；遗忘轴的NLL中位数与参考差异小于家族复制噪声\(\delta_f\)；保留损伤不超过\(M_{inj}\)的保留NLL加噪声容差；往返损伤不超过家庭往返地板值加噪声容差。

### 3. 实验设计
- **核心对比场景**：45个模型-种子单元格，覆盖5个开放架构家族（Qwen2.5-0.5B/1.5B、SmolLM2-360M/1.7B、Pythia-410M/1.4B、GPT-2 medium/large、OLMo-2-1B），包含SwiGLU和GELU变体。每个细胞有4个遗忘事实、4个保留事实和16个从未学习探针。
- **评估的遗忘方法**：路径特定KL回归、任务向量否定、gate\_proj掩码回归、带保留正则的梯度上升（GA）和负偏好优化（NPO），以及检查点回滚作为基线。
- **验证工具**：
  - **密封挑战套装**：包含基线模型、注入模型、参考模型、logit抑制攻击（遗忘答案logit减去10）、实体路由抑制、嵌入破坏等，用于测量筛查的灵敏度和假接受率。
  - **外部验证**：在TOFU数据集（Phi-1.5, forget10/retain90）上评估诊断效果和无参考选择边界。
- **指标**：训练探针KL（oracle-KL）、保留外遗忘NLL、保留NLL、往返残余、一般损害（wikitext困惑度）等。

### 4. 资源与算力
- 所有实验在**单块48 GB RTX 6000 Ada GPU**上完成。论文未报告总训练时长，但详细给出了各类过程的步数（注入200–250步，遗忘约110–120步，再获取75–80步等），整体算力需求相对较小，属于中小规模可控实验。

### 5. 实验数量与充分性
- **主要实验矩阵**：9配置×5种子×5家族，共45个单元格，训练了**820个遗忘候选模型**，并对每个细胞执行了往返再获取、筛查、挑战面板审计。
- **消融与审计**：
  - 对参考模型本身进行组件式审计（保留、往返、筛查），验证固定阈值是否合理。
  - 3次独立重训练以量化每个家族的复制噪声\(\delta_f\)（P90达0.82–2.92纳特）。
  - 在不同再获取预算（15,30,45,75步）下测量往返地板值的漂移。
  - 挑战面板包含7类模型（M0, Minj, oracle, logit抑制, 路由, 嵌入破坏, 插值），跨所有45细胞进行盲测。
- **公平性与客观性**：所有重新校准规则、阈值和审计池在分析前冻结并提交，分析脚本可复现；密封挑战集仅评分一次，确保了统计独立性。实验设计系统且审计严格，但小事实集（m=4）限制了统计功效，且跨家族复制噪声差异大。

### 6. 主要结论与发现
- 常用的**训练探针标准奖励残留知识**：其认为“合格”的候选，在保留外遗忘事实上的NLL比从未学习水平低-2.82 nats。
- **绝对认证阈值自身的参考模型都通不过**：固定保留阈值≤0.5和往返阈值≤0.5极其苛刻，注入模型自身在41/45细胞中未通过保留阈值，31/45细胞中未通过往返阈值；匹配参考模型仅在1/45细胞中完全认证。问题在于阈值低于任务工作点，且往返地板值由再获取预算漂移主导。
- **基线锚定的保留外筛查是有效的必要测试**：在45/45细胞中拒绝注入模型，44/45接受参考模型，并能部分检测实体路由（35/45），表现出“基线漂移接缝敏感度”。
- **仅前向的认证不健全**：固定幅度的logit抑制攻击在12/45细胞中欺骗了整个前向电池（上层Clopper-Pearson上限39.6%）。这说明前向电池无法应对行为对抗。
- **损害相对重新校准提供选择性部分阳性**：在15/45细胞中认证了38个候选（集中于噪声最大的家族），所选模型在优化轴上的性能处于重训练噪声范围内（0.80 nats），远优于训练探针标准所选模型的5.17 nats偏差。
- **无参考选择的边界由可识别性划定**：对于反事实（nonce）事实，无参考选择可行；对于可推断事实（如TOFU），遗忘阈值无法从未学习探针分布恢复，因此选择器应弃权（abstain），这与TOFU实验完全一致。

### 7. 优点
- **方法论革新**：将遗忘从“答案抑制”重新定义为“分布恢复”，并为之设计了一套完整的无参考评估与筛选框架。
- **严格的自我审计**：将提出的参考模型本身作为受试对象，检验其能否通过自己的标准，从而暴露了固定阈值的根本缺陷。
- **多维度验证**：通过密封挑战套装、多家族复现、外部TOFU验证，全面测量了筛查的灵敏度、假接受率、检测路由的能力以及对抗攻击的边界。
- **理论与实证结合**：给出了可识别性定理，从理论上划分了无参考认证的可能区域，并在TOFU上得到验证。
- **可复现性与透明性**：提供了密码学哈希密封审计池、冻结分析规则，并承诺开源代码和配置。

### 8. 不足与局限
- **威胁模型有限**：证书仅针对“生产出的方法”，对自适应双层攻击（针对往返残余优化）未测试；前向电池被logit抑制攻破说明其无法应对行为对抗者。
- **重校准的回顾性**：损害相对证书在已存储的候选上回顾性评估，缺少重新学习否决轴（relearn-veto），因为存储的候选没有重新学习轨迹。
- **事实集规模小**：每个细胞仅4个遗忘事实，虽筛查分辨率通过Holm校正要求，但统计功效对较小效应量有限。
- **复制噪声估计粗糙且集中**：每个家族的\(\delta_f\)仅基于3次重训练重抽样，且大部分认证来自噪声最大的Qwen家族（\(\delta_f=2.92\)），在保守全局阈值下认证候选数锐减。
- **模型与数据规模限制**：使用≤1.7B参数的小模型和合成临时事实，尚未在大型指令微调模型上验证，通用性受限。
- **bf16训练的非确定性**：所有参考衍生的数值都附带量化的复制噪声，增加了不确定性。
- **选择的无参考性局限于反事实事实**：在可推断事实场景下（如TOFU）仍需oracle或弃权。
