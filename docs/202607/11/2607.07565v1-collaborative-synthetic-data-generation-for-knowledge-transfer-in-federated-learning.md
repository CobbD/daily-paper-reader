## 摘要
单次联邦学习（OSFL）通过将训练限制为单轮来解决联邦学习的通信开销问题，但在不牺牲模型质量的前提下做到这一点并非易事，尤其当客户端数据分布存在差异时。近期工作通过在服务器端构建可迁移的合成数据集或蒸馏物来聚合客户端知识，从而应对这一挑战。然而，这些方法大多缺乏正式的隐私保证，在同时实现低通信、对异构性的鲁棒性以及严格的隐私保护方面存在不足。我们提出FedKT-CSD（通过协作式合成数据进行联邦知识迁移），这是一个受神经图像压缩启发的框架，通过利用公开预训练的自编码器作为共享潜在空间来弥补这一不足。每个客户端在单次前向传递中对其私有数据进行编码，计算类别条件潜在统计量，并将其传输至服务器。服务器通过安全聚合来聚合这些统计量，添加经过校准的差分隐私噪声，并解码出一个合成数据集，用于训练全局模型及后续的下游任务。该设计在构造上提供了形式化的$(\varepsilon,\delta)$-差分隐私，同时保持客户端计算和通信的轻量化。尽管在隐私约束下运行，FedKT-CSD在各种数据集和异构性设定下仍能与非私有基线方法相媲美，甚至更优，并且可扩展至大量客户端。我们的代码可在以下地址获取：https://github.com/an7123/FedKT-CSD

## Abstract
One-shot federated learning (OSFL) addresses the communication overhead of federated learning by limiting training to a single round, but doing so without sacrificing model quality is non-trivial, particularly when client data distributions diverge. Recent work has addressed this challenge by aggregating client knowledge on the server through the construction of transferable synthetic datasets or distillates. However, most of these methods lack formal privacy guarantees, leaving a gap in jointly achieving low communication, robustness to heterogeneity, and rigorous privacy. We propose FedKT-CSD (Federated Knowledge Transfer via Collaborative Synthetic Data), a framework inspired by neural image compression that closes this gap by leveraging publicly pretrained autoencoders as a shared latent space. Each client encodes its private data in a single forward pass, computes class-conditional latent statistics, and transmits these to the server. The server aggregates these statistics via secure aggregation, adds calibrated differential privacy noise, and decodes a synthetic dataset for training a global model and further downstream tasks. This design provides formal $(\varepsilon,δ)$-differential privacy by construction, while keeping client-side computation and communication lightweight. Despite operating under privacy constraints, FedKT-CSD is competitive with and even outperforms non-private baselines across diverse datasets and heterogeneity settings, and scales to a large number of clients. Our code is available at: https://github.com/an7123/FedKT-CSD

*   **目标**：设计一个框架，既能利用 OSFL 中的表征共享优势，又能提供形式化隐私保证和轻量级客户端计算。

## 2. 方法论：FedKT-CSD
*   **核心思想**：利用公开预训练的神经图像压缩自编码器（如 DC-AE）构建一个**共享的潜在空间**。客户端仅需一次前向传播编码私有数据，计算每类的潜在统计量（总和、外积和、样本数），服务器通过安全聚合和差分隐私噪声合成全局数据集。
*   **关键技术细节**：
    *   **客户端编码**：
        1. 使用冻结的编码器 $E$ 将图像 $x$ 映射到潜在向量 $z = E(x)$。
        2. 对 $z$ 进行 $\ell_2$-范数裁剪至 $R$，得到 $\bar{z}$，以保证敏感度有界。
        3. 对每个类 $c$，计算三个可加统计量：潜在向量和 $\mathbf{m}_i^c = \sum \bar{z}_{i,j}$，外积和 $\mathbf{M}_i^c = \sum \bar{z}_{i,j} \bar{z}_{i,j}^\top$，样本计数 $n_i^c$。
    *   **安全聚合与加噪（服务器端）**：
        1. 通过安全聚合协议（如 SecAgg）对各客户端上传的统计量求和，得到全局总和。
        2. 依据**解析高斯机制**添加校准过的差分隐私噪声（对总和和外积和分别添加，满足 $(\varepsilon, \delta)$-DP）。
    *   **后处理与合成**：
        1. 恢复有噪均值 $\mu_c^{DP}$ 和有噪协方差矩阵 $\widehat{\Sigma}_c$。
        2. 进行**偏差校正**（因均值噪声导致外积减量偏差）和**半正定投影**（确保协方差矩阵可采样）。
        3. 从生成的高斯分布 $\mathcal{N}(\mu_c^{DP}, \Sigma_c^{DP})$ 中采样潜在向量，并用冻结的解码器 $D$ 解码为像素空间的合成图像数据集 $X^*$。
*   **隐私分析要点**：
    *   采用**类内替换相邻关系**（类标签公开）。
    *   敏感度：和向量为 $\Delta_\mu = 2R$，外积和为 $\Delta_\Sigma = 2R^2$。
    *   总隐私预算 $(\varepsilon, \delta)$ 平分给两个统计量，通过解析高斯机制校准噪声，满足组合定理。
    *   所有后处理（归一化、校正、采样、解码、下游训练）均不消耗额外隐私预算。

## 6. 主要结论与发现
*   **DP 下性能优越**：FedKT-CSD 在满足 $(\varepsilon=10, \delta=10^{-5})$-DP 的情况下，在全部数据集和异质性设置上**均优于所有非隐私的 OSFL 基线**。
*   **异构性不变性**：由于聚合的是加性统计量而非模型参数，方法性能对客户端数据异质程度**完全不变**，这是现有模型聚合方法无法实现的。
*   **强可扩展性**：客户端数量从 20 扩展至 1000 时，性能几乎无衰减，远优于随客户端增多而急剧退化的基线方法。
*   **下游任务通用性**：生成的像素空间合成数据集可作为预训练语料，在多轮个性化联邦学习中仅凭单轮通信即达到或超越 SOTA pFL 方法的性能。
*   **极高效率**：客户端无需训练，计算耗时秒级，上传量 KB 级，适合资源受限设备。

## 7. 优点与亮点
*   **隐私设计优雅**：利用冻结公共自编码器的低维潜在空间，将复杂的生成任务转化为低维统计量的加噪聚合，隐私分析与实现天然契合。
*   **无客户端训练**：彻底避免客户端优化、梯度计算和模型传输，极大降低计算与通信成本。
*   **异构性鲁棒性天然内置**：通过统计量加法，将异构数据分布问题解耦，确保全局分布重构的精确性。
*   **输出通用性**：合成的是像素级图像，非定域于特定特征提取器，可复用为任意架构的数据集，应用场景广泛。
*   **理论与实践并重**：提供了清晰的敏感度推导、隐私组合证明以及大量的经验验证，附录还讨论了扩展到完整添加/删除记录级别 DP 的路径。

## 8. 不足与局限
*   **强标签依赖**：方法假设为有监督分类任务，且类别标签在客户端已知并公开（附录提及可通过加噪计数扩展，但主实验未包含），无法直接用于无监督或自监督场景。
*   **建模能力受限**：假设每类潜在空间服从单高斯分布，可能无法充分捕捉复杂或高度重叠的类内分布（如考虑高斯混合模型 GMM 会破坏加性聚合特性）。
*   **小样本类噪声**：当某类别总体样本数 $n_c$ 很小时，DP 噪声相对增强可能导致该类的统计量质量下降。
*   **预训练自编码器依赖**：合成图像质量受限于所用自编码器的重建能力，对域外数据（如医学、卫星图）的性能可能弱于自然图像。
*   **仅保护特征隐私**：主实验采用类内替换相邻定义，保护样本特征值但不保护类别参与与否（已通过替换为添加/删除相邻进行了分析，但未实验验证）。
