## 摘要
一次性联邦学习（OSFL）通过将训练限制在单轮内来解决联邦学习的通信开销问题，但在不牺牲模型质量的前提下实现这一目标并非易事，特别是当客户端数据分布存在差异时。近期工作通过构建可迁移的合成数据集或蒸馏物在服务器上聚合客户端知识来应对这一挑战。然而，这些方法大多缺乏正式的隐私保证，在同时实现低通信、对异质性的鲁棒性以及严格隐私方面存在差距。我们提出了FedKT-CSD（通过协作合成数据进行联邦知识迁移），这是一个受神经图像压缩启发的框架，通过利用公开预训练的自编码器作为共享潜在空间来弥补这一差距。每个客户端在单次前向传播中对其私有数据进行编码，计算类别条件的潜在统计量，并将这些统计量发送给服务器。服务器通过安全聚合汇总这些统计量，添加校准的差分隐私噪声，并解码出一个合成数据集，用于训练全局模型以及进一步的下游任务。该设计通过构造提供了正式的$(\varepsilon,\delta)$-差分隐私，同时保持客户端计算和通信的轻量级。尽管在隐私约束下运行，FedKT-CSD 在多种数据集和异质性场景下与无隐私基线方法相比具有竞争力，甚至表现更优，并且能够扩展到大量客户端。我们的代码可在以下链接获取：https://github.com/an7123/FedKT-CSD

## Abstract
One-shot federated learning (OSFL) addresses the communication overhead of federated learning by limiting training to a single round, but doing so without sacrificing model quality is non-trivial, particularly when client data distributions diverge. Recent work has addressed this challenge by aggregating client knowledge on the server through the construction of transferable synthetic datasets or distillates. However, most of these methods lack formal privacy guarantees, leaving a gap in jointly achieving low communication, robustness to heterogeneity, and rigorous privacy. We propose FedKT-CSD (Federated Knowledge Transfer via Collaborative Synthetic Data), a framework inspired by neural image compression that closes this gap by leveraging publicly pretrained autoencoders as a shared latent space. Each client encodes its private data in a single forward pass, computes class-conditional latent statistics, and transmits these to the server. The server aggregates these statistics via secure aggregation, adds calibrated differential privacy noise, and decodes a synthetic dataset for training a global model and further downstream tasks. This design provides formal $(\varepsilon,δ)$-differential privacy by construction, while keeping client-side computation and communication lightweight. Despite operating under privacy constraints, FedKT-CSD is competitive with and even outperforms non-private baselines across diverse datasets and heterogeneity settings, and scales to a large number of clients. Our code is available at: https://github.com/an7123/FedKT-CSD

- **核心思想**：将预训练的神经压缩自编码器（如 DC-AE）作为所有客户端共享的、冻结的编码器/解码器，在隐空间中以可加性统计量（类条件均值的求和、二阶矩求和、样本计数）替代模型参数或原始数据，通过安全聚合消除客户端划分的影响，再在聚合统计量上注入校准的高斯噪声实现差分隐私，并从 DP 保护的类条件高斯分布中采样、解码合成图像，用于下游模型训练。
- **端侧流程（单次前传，无训练）**：
  - 客户端将私有图像通过冻结编码器 \(E\) 编码为隐向量 \(z_{i,j}\)，并按 ℓ2 范数裁剪至半径 \(R\)：\(\bar{z}_{i,j} = \mathrm{Clip}_R(z_{i,j})\)。
  - 对每个类别 \(c\) 计算三个可加统计量：隐向量和 \(m_{i}^c = \sum_j \bar{z}_{i,j}\)，外积和 \(M_{i}^c = \sum_j \bar{z}_{i,j} \bar{z}_{i,j}^\top\)，以及样本数量 \(n_{i}^c\)。
  - 将三元组 \((m_i^c, M_i^c, n_i^c)\) 上传至服务器（通过安全聚合协议，服务端仅获取全局总和）。
- **服务端隐私与合成**：
  - 全局聚合得到 \(S_{\mu,c} = \sum_i m_i^c\)、\(S_{\Sigma,c} = \sum_i M_i^c\)、\(n_c = \sum_i n_i^c\)。
  - 按照分析高斯机制，分别对 \(S_{\mu,c}\)、\(S_{\Sigma,c}\) 注入独立高斯噪声（噪声标准差由敏感性 \(2R\)、\(2R^2\) 及隐私预算 \(\varepsilon,\delta\) 校准）。

### 7. 优点（方法与实验设计的亮点）
- **隐私即设计**：通过冻结的公开编码器、加性统计量、安全聚合和一次性噪声注入，实现了自然、紧凑的 \((\varepsilon,\delta)\)-DP 保证，无需繁杂的隐私会计。
- **低敏感性空间**：在低维隐空间（d=128）上计算统计量，相较于高维模型参数或特征，噪声引入的总能量较低，信噪比优势显著，因而在中等隐私预算下仍维持高效用。
- **异质性与规模不变**：可加性聚合从根本上避免了模型聚合中的客户端漂移，天然支持任意客户端划分和数量扩展。
- **彻底的客户端轻量化**：零训练、极低通信负载（数百 KB），对边缘设备友好。
- **生成的像素级合成数据**：生成的不是固定特征，而是可复用的图像，大幅拓宽了下游应用场景（架构灵活、预训练、知识蒸馏、数据增强、多方共享等）。
- **实验全面扎实**：覆盖多域数据集、剧烈异质性、多种对比方法（OSFL、DP 合成、pFL）、丰富的消融和隐私审计，结论可信度高。

### 8. 不足与局限
- **标签公开假设**：主要方案假定类别标签公开（隐私仅保护特征值），虽然补充材料提供了隐藏类别参与（如加噪计数）的扩展，但未给出其详细性能评估。
- **单高斯建模限制**：每个类别仅以单一高斯分布概括，在类内分布复杂或多模态时表达能力有限，可能影响生成质量。
- **小样本类噪声放大**：当某类样本数 \(n_c\) 很小时，噪声在均值中的影响剧烈（\(\sigma_\mu/n_c\) 变大），这属于 DP 机制的固有挑战。
- **依赖预训练编码器**：合成质量依赖于公开自编码器的领域匹配度；对卫星、医学等非自然图像，虽仍可工作，但编码器可能不是最优，泛化性受限于预训练数据的分布。
- **仅适用于有监督分类**：需要客户端的样本带有类别标签，未涉及无监督或自监督场景。
- **部分运算资源未交代**：服务器端模型训练（如 ResNet-18 训练合成数据）及多轮 pFL 实验的详细算力未予说明。
- **威胁模型范围**：文中采用“诚实但好奇”的服务器模型，并假设安全聚合后服务器正确添加 DP 噪声上传；未深入探讨恶意服务器或更强敌手场景。
