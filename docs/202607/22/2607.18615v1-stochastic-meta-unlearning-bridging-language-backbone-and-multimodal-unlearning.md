## 摘要
视觉语言模型（VLMs）的机器反学习仍处于探索不足的状态。与语言模型不同，VLMs将语言骨干与视觉组件相结合，这使得反学习更为复杂。当从单模态反学习转向VLM反学习时，存在一个令人惊讶的现象：被独立语言骨干遗忘的目标在给定图像信息给完整VLM时仍能被恢复。这表明仅靠文本反馈不足以实现可靠的VLM反学习。受此观察启发，我们提出了随机元反学习（SMU），这是一个双层框架，利用VLM级别的反馈来学习准备好反学习的初始化。在内循环中，SMU使用文本数据对语言骨干应用少量的反学习步骤。在外循环中，SMU将更新后的骨干与冻结的VLM重新组合，并在VLM级别评估遗忘和效用。这种设计使反学习更新能够感知最终的多模态行为，同时仍将更新保持在语言骨干局部。在两个VLMs、两个多模态模因数据集和三个基线方法上的实验表明，SMU实现了最佳的遗忘-保留平衡。与每个指标上的最强基线相比，SMU将平均遗忘准确率降低了10.52个百分点，并将平均保留和测试准确率分别提高了20.10和17.01个百分点。更重要的是，SMU还能迁移到新的遗忘目标和不同的元测试反学习方法。这些结果表明，VLM级别的反馈可以使语言骨干反学习对于VLMs更加可靠和更具迁移性。

## Abstract
Machine unlearning for vision-language models (VLMs) remains underexplored. Unlike language models, VLMs combine a language backbone with visual components, which makes unlearning more complex. There is a surprising phenomenon when moving from single-modality unlearning to VLM unlearning: a target forgotten by the standalone language backbone can still be recovered when image information is given to the full VLM. This shows that text-only feedback is not enough for reliable VLM unlearning. Motivated by this observation, we propose Stochastic Meta-Unlearning (SMU), a bilevel framework that uses VLM-level feedback to learn an unlearning-ready initialization. In the inner loop, SMU applies a few unlearning steps to the language backbone using text data. In the outer loop, SMU recomposes the updated backbone with the frozen VLM and evaluates forgetting and utility at the VLM level. This design makes the unlearning update aware of the final multimodal behavior, while still keeping the update local to the language backbone. Experiments on two VLMs, two multimodal meme datasets, and three baselines show that SMU achieves the best overall forget-retain trade-off. Compared with the strongest baseline for each metric, SMU reduces average Forget accuracy by 10.52 points and improves average Retain and Test accuracy by 20.10 and 17.01 points, respectively. More importantly, SMU also transfers to new forgetting targets and to different meta-test unlearning methods. These results suggest that VLM-level feedback can make language-backbone unlearning more reliable and more transferable for VLMs.

### 2. 方法论：随机元反学习（SMU）

- **核心思想**：将反学习过程分为内外两个循环，通过元训练学习一种语言骨干初始化，使得从该初始化出发，仅需几次文本侧的反学习步骤，重组后的VLM即可在文本和多模态条件下都遗忘目标，同时保持保留集的性能。
- **内循环（语言骨干反学习）**：在采样的特定遗忘目标、反学习算子等环境下，对语言骨干参数 θT 执行 K 步文本侧反学习更新，例如：
  
  θ_{T,k+1} = θ_{T,k} - α ∇ L_in( f_{θ_{T,k}} ; B_f, B_r )
  
  这里只使用文本遗忘/保留样本，不涉及图像或视觉模块。
- **外循环（VLM级反馈）**：将经内循环更新后的骨干与冻结的视觉组件 ϕ 重组为 F_{θ_{T,K},ϕ}，在遗忘图像‑文本对、保留图像‑文本对以及纯文本查询上计算外层损失：
  
  L_outer = λ_f L_mm_f + λ_r L_mm_r + λ_text_f L_text_f + λ_text_r L_text_r
  
  该损失通过重组VLM的输出衡量遗忘的彻底性和保留的完整性，其梯度通过内循环的 K 步展开回传，用于更新初始化参数 θT。
- **随机采样与泛化**：为避免过拟合到固定环境，SMU 从环境分布 p(E) 中随机采样不同的遗忘目标 z、内循环反学习算子 m 和查询条件 c，在每个元训练批次中平均多个环境的损失，从而学到可迁移的初始化。
- **元测试**：得到元学习初始化 θ*_T 后，面对新的遗忘目标或新的反学习算子，只需在文本侧进行少量反学习步骤，再将更新后的骨干与 ϕ 重组，即可部署。

### 6. 主要结论与发现

- **SMU 在遗忘-保留权衡上全面优于直接VLM反学习基线**：相比最强基线，平均遗忘准确率降低 10.52 个百分点，保留准确率提高 20.10 个百分点，测试准确率提高 17.01 个百分点。GradDiff 等直接优化虽有时遗忘更多，但会严重破坏保留能力。
- **VLM级反馈解决组件-系统不匹配**：纯文本遗忘不足以保证多模态遗忘，而 SMU 通过在外循环评估重组VLM，迫使初始化学习到对多模态遗忘敏感的轨迹。
- **元初始化可迁移**：SMU 能在新的遗忘目标上保持低遗忘率，且当元测试时的反学习算子（如从 GradDiff 换成 RMU）发生变化时仍有效，验证了初始化不是对特定算子过拟合。
- **迁移性来自充分元训练**：遗忘准确率在元训练早期可能上升，后期下降，保留能力始终稳定，说明足够的元训练对于获得可迁移初始化至关重要。

### 7. 优点

- **巧妙的双层框架**：将“局部更新”与“全局多模态验证”结合，在不更新视觉模块的前提下，用重组VLM的反馈指导语言骨干的预更新状态，解决了单模态遗忘不可靠的难题。
- **计算高效且模块友好**：实际反学习（元测试）只进行文本侧轻量更新，视觉编码器和投影层冻结，避免了多模态训练开销，也降低了对齐破坏风险。
- **随机环境采样增强泛化**：通过采样不同的遗忘目标、算子和条件进行元训练，使初始化具备对多变遗忘任务的迁移能力，提升实用价值。
- **评估全面**：同时测量文本侧和多模态侧的遗忘/保留，揭示了单模态遗忘的假象，评测协议更贴近实际部署。
- **实验扎实**：在两个主流VLM和两个多模态数据集上验证，与多个强基线比较，并考察目标迁移和算子迁移，证据较充分。

### 8. 不足与局限

- **元训练成本**：需展开内循环并计算高阶梯度，元训练阶段比单次反学习更耗时，且需存储计算图，对资源有一定要求。
- **视觉组件冻结的限制**：当需要遗忘的信息主要编码在视觉编码器或投影层时，仅更新语言骨干可能无法彻底移除知识；该方法未处理视觉侧的遗忘需求。
